# The Talos Manifesto

> **"We build the machine that builds the machine."**

## 1. The Vision: The Cyborg Company
We are not a traditional company. We are a **Human-Machine Business**.
*   **The Captain (Human)**: Sets the **Intent** (Strategy) and **Taste** (QA).
*   **The Crew (Agents)**: Provides the **Velocity** (Execution) and **Scale** (Operations).

**The goal**: High-Leverage Human Time. The system succeeds when the human spends 100% of their time on Strategy and Taste, and 0% on "Boring Tasks."

## 2. The Core Values

### 1. Truth over Comfort
We do not lie to ourselves. Data matters. If the metrics say we failed, we failed. We do not use "corporate speak" to hide reality.

### 2. Velocity is Quality
Speed is not the enemy of quality; it is the prerequisite. Taking 3 months to ship a feature is a failure of architecture, not a badge of thoroughness.

### 3. Radical Simplicity (Zero Entropy)
Complexity is the silent killer. We fight it with fire.
*   No new tool unless the old one is broken.
*   No new meeting unless an email failed.
*   No new code unless a library doesn't exist.
*   If we build something, we intent to make it reusable.
*   No new agent unless the old one is broken.

## 3. The First Principles (The Physics)
1.  **Gravity (Security)**: If it's not secure, it's a liability. We harden first.
2.  **Energy (Compute)**: Optimize for Signal, not token count. Better to run a small model twice than a big model once.
3.  **Mass (Context)**: Context rot is the enemy. Keep agents lightweight and sane.

## 4. The Non-Negotiables
1.  **AI-First**: If a human is doing verifyable, repeatable work, automate it.
2.  **Open Books**: Information cannot be siloed. `2_context/SYSTEM_MAP.md` is available to everyone.
3.  **Taste**: We do not ship "MVP" garbage. We ship "Minimum Lovable Products".
