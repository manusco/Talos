# ❓ Frequently Asked Questions (FAQ)

## 🎯 Onboarding

### Why do I need `talos/`? Can't I just chat?
Most AI chats are "amnesiac". Talos gives your agent a **Hard Drive**. The `talos/` folder stores your Strategy and Memory so the Agent gets smarter over time.

### How do I update the rules?
Edit `1_strategy/MANIFESTO.md`. This is the "Soul". Whatever you write there is Law for every Agent that touches this project.

### Can I run a system check manually?
Yes.
**Mac / Linux**: `./talos.sh`
**Windows (PowerShell)**: `.\talos.ps1`

### I have my own way of doing things. Can I change the workflows?
Yes. Workflows in `.agent/workflows/` are just markdown files. Edit them to fit your team's style. Resonance is descriptive, not prescriptive.
